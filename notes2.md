# Module 2: Variables

## Overview
Variables are used to store data in a program. They help you save values and use them later.

---

## What is a Variable?

A variable is a name that stores a value.

---

## Creating Variables

```python
x = 10
name = "Hemant"
```

- `x` stores a number  
- `name` stores text  

---

## Printing Variables

```python
x = 10
name = "Hemant"

print(x)
print(name)
```

Output:

```
10
Hemant
```

---

## Variable Types

```python
a = 10        # Integer
b = 3.14      # Float
c = "Hello"   # String
d = True      # Boolean
```

---

## Rules for Naming Variables

- Must start with a letter or underscore  
- Cannot start with a number  
- No spaces allowed  
- Use meaningful names  

Examples:

```python
age = 25
student_name = "Ravi"
```

---

## Changing Variable Value

```python
x = 5
x = 10

print(x)
```

Output:

```
10
```

---

## Practice

```python
name = "Student"
age = 20

print(name)
print(age)
```
