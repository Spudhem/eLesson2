# Module 1: Introduction to Python

## Overview
Python is a simple and easy-to-learn programming language. It is widely used in bioinformatics, data analysis, and automation.

---

## What is Python?

- Python is a programming language  
- It is easy to read and write  
- Used in research, data science, and biology  

---

## Running Python

### Start Python interpreter

```bash
python
```

You will see something like:

```
>>>
```

---

### Exit Python

```bash
exit()
```

---

## Your First Python Program

```python
print("Namskar!")
```

Output:

```
Namskar!
```

---

## Understanding print()

- `print()` is used to display output  
- Text should be written inside quotes `" "`
